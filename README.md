# 🪨📄✂️ Stone, Paper, Scissors

A simple, interactive command-line implementation of the classic "Stone, Paper, Scissors" game built entirely in Python. 

## 📖 Description

This project is a beginner-friendly Python script where you play against a computer opponent. The computer uses randomized logic to make its choices, and the game keeps track of the score across multiple rounds until you decide to quit.

## ✨ Features

* **Infinite Gameplay:** Play as many rounds as you want in a single session.
* **Score Tracking:** Keeps a running tally of both the player's and the computer's score.
* **Input Validation:** Handles typos gracefully without crashing the program.
* **Clean Interface:** Simple and easy-to-read command-line outputs.

## 🛠️ Prerequisites

To run this game, you will need to have Python installed on your computer. 
* Python 3.6 or higher is recommended.

## 🚀 How to Run

1. Save the game code to a file named `game.py`.
2. Open your terminal or command prompt.
3. Navigate to the folder where you saved `game.py`.
